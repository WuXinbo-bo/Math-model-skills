# 论文格式模板库

> 存放高分获奖论文 PDF，供 S8/S9/S9.5 阶段的子 Agent 直接读取参考。

## 使用方式

将高分论文 PDF 放入本文件夹即可。子 Agent 在撰写/审稿/出版审查时，直接用 `read_file` 读取 PDF，自行提取格式模式作为参考。

```
templates/paper_templates/
├── README.md          # 本文件
├── A196.pdf           # 高分论文
├── B060.pdf
├── B157.pdf
├── C023.pdf
├── C132.pdf
├── D037.pdf
└── E030.pdf
```

## 引用方式

- **S8 撰写**：Writer 子 Agent 读取 1-2 篇 PDF，参考其章节结构、图表排布、公式编号等格式模式
- **S9 审稿**：Critic/Reviewer 对照 PDF 检查论文格式合规性
- **S9.5 出版审查**：对照 PDF 做最终格式合规校验

## 原则

- 只参考**格式结构**（章节标题、表格格式、图表排布），不要求内容与参考论文一致
- 可随时添加新 PDF，无需额外操作
