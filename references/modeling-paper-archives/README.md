## 使用指南

### 如何阅读论文

```bash
# 使用 pdf skill 读取 PDF
# 1. 确定需要参考的论文类型
# 2. 读取对应的 PDF 文件
# 3. 提取关键方法和写作技巧
```

### 学习重点

| 维度 | 关注点 |
|------|--------|
| 论文结构 | 章节安排、逻辑顺序、过渡衔接 |
| 摘要写作 | 四要素齐全、方法名具体、数值精确 |
| 模型表述 | 公式规范、符号定义、推导完整 |
| 图表设计 | 类型选择、标注规范、D-A-C 分析 |
| 结果分析 | 深入解读、基线对比、灵敏度分析 |
| 验证检验 | 鲁棒性、误差分析、极端场景 |

### 注意事项

- 仅供学习参考，请勿直接复制
- 理解方法后用自己的方式表达
- 论文格式可能随年份有所变化
- 重点关注方法论而非具体数值
