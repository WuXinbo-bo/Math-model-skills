# CUMCM execution profile

Use the bundled `cumcmthesis` template and the current official CUMCM formatting notice. The electronic paper body must remain anonymous, contain Chinese abstract and keywords, use traceable citations, and pass the current-year page/attachment/submission requirements supplied by the organizing committee.

For the 2026 profile, omit the table of contents, keep the abstract within one page, keep the body within 30 pages, and count the appendix separately. Preserve the `AbstractStart/AbstractEnd` and `BodyStart/BodyEnd` labels so the gate can measure compiled pages. Generate the appendix from `程序/code_manifest.json`; include the support-material file list and all complete runnable source programs.
