# CUMCM（国赛）评分标准详解

> 适用阶段：S1 选题决策、S9 论文撰写、S10 终审
> 用途：Agent 写论文时对标评分维度，确保不丢分

---

## 一、评分维度与权重（2023-2025 评阅要点综合）

| 维度 | 权重 | 评委关注点 |
|------|------|-----------|
| **模型质量** | 50-60% | 模型的合理性、创新性、深度；数学推导是否严谨 |
| **问题解决** | 20-30% | 是否准确理解题意；是否逐题回答；结论是否清晰 |
| **论文规范** | 10-15% | 摘要质量、格式规范、语言表达、图表质量 |
| **验证与检验** | 5-10% | 结果验证、灵敏度分析、鲁棒性检验 |

### 创新性加分（额外 5-10 分）
- 模型创新：提出新方法或对已有方法的有意义改进
- 方法组合：将不同领域方法创新性地组合
- 视角独特：从非常规角度切入问题

### 扣分重灾区
| 扣分项 | 典型扣分 | 说明 |
|--------|---------|------|
| 摘要缺失关键信息 | -5~10 | 无方法、无结果、无结论 |
| 图表无正文解读 | -5~8 | 图表堆砌无 D-A-C 分析 |
| 结论与正文矛盾 | -10~15 | 摘要写"精度99%"正文只有85% |
| 格式严重不规范 | -5~10 | 字体混乱、页边距不统一、无页码 |
| 无灵敏度/鲁棒性分析 | -5~8 | 直接出结论无验证 |
| 编造数据/参考文献 | -15~20 | 直接取消评奖资格 |

## 二、各分数段特征

### 一等奖（Top 5-10%）
| 维度 | 要求 |
|------|------|
| 模型 | 有明确创新点，数学推导严谨完整，方法选择有理有据 |
| 论文 | 摘要精炼（300-500字覆盖全貌），逐题回答，图表有D-A-C解读 |
| 验证 | 完整的灵敏度分析 + 鲁棒性检验 + 基线对比 |
| 写作 | 专业排版，图表美观，语言简洁，无废话 |

### 二等奖（Top 10-25%）
| 维度 | 要求 |
|------|------|
| 模型 | 合理的模型构建，推导基本完整，方法选择合理 |
| 论文 | 摘要基本完整，逐题回答，有图表解读 |
| 验证 | 有基本的灵敏度分析，部分结论有验证 |
| 写作 | 格式基本规范，图表可读 |

### 三等奖（Top 25-50%）
| 维度 | 要求 |
|------|------|
| 模型 | 基本合理的模型，推导不够深入 |
| 论文 | 摘要缺失部分信息，回答基本完整 |
| 验证 | 灵敏度分析简单或缺失 |
| 写作 | 格式有瑕疵，图表不够规范 |

### 未获奖（Bottom 50%）
| 维度 | 典型问题 |
|------|---------|
| 模型 | 跑题、模型与问题不匹配、无推导 |
| 论文 | 摘要过短/过长、未逐题回答、结论缺失 |
| 验证 | 无任何验证 |
| 写作 | 格式混乱、图表无法理解 |

## 三、逐维度评分细则

### 3.1 模型质量（50-60%）

#### 模型构建（30%）
| 等级 | 分数 | 标准 |
|------|------|------|
| 优秀 | 9-10 | 模型创新且推导严谨，假设合理且标注必要性 |
| 良好 | 7-8 | 模型合理，推导基本完整 |
| 及格 | 5-6 | 模型基本可行，推导有缺失 |
| 不及格 | <5 | 模型不匹配或无推导 |

#### 模型选择与比较（15%）
| 等级 | 分数 | 标准 |
|------|------|------|
| 优秀 | 4-5 | 说明了备选方案和选择理由，有消融实验 |
| 良好 | 3 | 说明了选择理由，无对比 |
| 不及格 | <3 | 未说明为什么选这个模型 |

#### 模型创新性（15%，加分项）
| 等级 | 标准 |
|------|------|
| 显著创新 | 提出新模型/新方法，有数学证明支撑 |
| 有意义改进 | 在已有方法基础上做了有理论支撑的改进 |
| 有限创新 | 简单套用已有方法，但适配合理 |
| 无创新 | 直接套用，无改进无解释 |

### 3.2 问题解决（20-30%）
| 等级 | 分数 | 标准 |
|------|------|------|
| 优秀 | 9-10 | 准确理解所有子问题，逐题有清晰答案，答案间逻辑一致 |
| 良好 | 7-8 | 理解正确，大部分有答案，少量遗漏 |
| 及格 | 5-6 | 基本理解，部分答案不完整 |
| 不及格 | <5 | 理解偏差或大量遗漏 |

### 3.3 论文规范（10-15%）
| 子项 | 评分要点 |
|------|---------|
| 摘要（5%） | 问题+方法+结果+结论四要素齐全，字数300-500字 |
| 格式（3%） | A4纸、页边距2.5cm、字体统一、页码正确、无空行 |
| 语言（3%） | 专业简洁，无口语化表达，无错别字 |
| 图表（4%） | 每图有标题+正文解读，图表编号连续，分辨率≥300DPI |

### 3.4 验证与检验（5-10%）
| 等级 | 分数 | 标准 |
|------|------|------|
| 优秀 | 4-5 | 完整的灵敏度分析 + 鲁棒性检验 + 基线对比 + 极端场景 |
| 良好 | 3 | 有基本灵敏度分析 |
| 不及格 | <3 | 无验证或仅简单测试 |

---

## 四、评委评审流程（内部参考）
```
1. 初审（5分钟/篇）：看摘要→判断是否跑题；看结论→判断是否有实质结果；看图表→判断工作量
2. 细审（20-30分钟/篇）：通读全文，关注模型构建和推导；检查图表是否有正文解读；核对摘要数值与正文是否一致
3. 终审（10分钟/篇）：对比同组论文排名；检查格式规范；最终打分
```
