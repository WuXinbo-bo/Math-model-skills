# Paper layout core

Load this file in MANUSCRIPT and ASSURANCE before format-specific rules.

1. Treat the competition page limit as a ceiling, never as a target to fill.
2. Preserve the scientific chain for every question: mechanism and variables, derivation or algorithm, validated result, interpretation, and limitation.
3. Reduce page pressure in this order: crop visual whitespace, resize visuals to the smallest readable size, combine redundant visuals, shorten captions, summarize long tables, move secondary evidence to appendices, remove repeated prose.
4. Never reduce essential derivation, validation, or result interpretation merely to save pages.
5. Never enlarge figures, line spacing, fonts, blank space, or generic prose to make a paper appear longer.
