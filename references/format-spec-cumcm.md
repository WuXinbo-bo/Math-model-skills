# 全国大学生数学建模竞赛 (CUMCM) 论文格式规范

基于《全国大学生数学建模竞赛论文格式规范（2025年修订稿）》及各赛区通行惯例。
所有 DOCX 生成必须严格遵循此规范。

## 官方硬性要求（必须遵守）

### 页面布局
| 项目 | 要求 |
|------|------|
| 纸张 | 白色 A4 纸 |
| 打印 | 单面、双面均可 |
| 页边距 | 上、下、左、右各至少 **2.5 厘米** |
| 装订 | 从**左侧**装订 |

### 论文结构（按页）
| 页序 | 内容 | 说明 |
|------|------|------|
| 第1页 | 承诺书 | 使用官方模板格式，不超过1页 |
| 第2页 | 编号专用页 | 使用官方模板格式 |
| 第3页 | 摘要专用页 | 含标题和关键词，**无需翻译成英文**，内容不能超过一页 |
| 第4页起 | 正文 | **不要目录**，尽量控制在 **20 页以内** |
| 正文后 | 附录 | 页数不限，必须打印并与正文装订在一起 |

### 页码规范
- 从摘要页（第3页）开始编写，位于**页脚中部**
- 阿拉伯数字从 "1" 开始连续编号

### 匿名要求（极其重要）
论文的**摘要页、正文和附录**中任何地方都**不能**出现：参赛者身份信息、所在学校信息、所在赛区信息。

### 参考文献要求
- 所有引用他人或公开资料的成果，必须按**科技论文规范**列出参考文献
- 必须在**正文引用处予以标注**

### 附录内容要求
1. 支撑材料的文件列表
2. 建模所用到的全部完整、可运行的**源程序代码**（含 EXCEL、SPSS 等软件的交互命令）
3. 如果确实没有用到程序，应在附录中明确说明："本论文没有用到程序"

### AI 工具使用规定
- 须遵守《全国大学生数学建模竞赛人工智能工具使用规定》
- 在附录中声明使用的 AI 工具名称、版本、用途
- 格式：工具名称与版本 / 使用目的与环节 / 使用方式说明

### 提交要求
| 项目 | 要求 |
|------|------|
| 提交文件数 | 两个文件：①参赛论文 ②支撑材料 |
| 论文格式 | PDF 或 Word（**建议使用 PDF**） |
| 文件大小 | 不超过 **20MB** |
| 是否压缩 | 论文文件**不要压缩** |
| 电子版首页 | 必须为**摘要专用页**（承诺书和编号页**不要**放入电子版）|
| 内容一致性 | 电子版必须与纸质版内容及格式完全一致 |

## 通用排版规范（官方未强制，但各赛区/评委通行惯例）

### 字体规范
| 元素 | 字体 | 字号 | 对齐 | 加粗 |
|------|------|------|------|------|
| 论文题目 | 黑体 | 三号 (16pt) | 居中 | 是 |
| 一级标题 | 黑体 | 四号 (14pt) | 居中 | 是 |
| 正文汉字 | 宋体 | 小四 (12pt) | 两端对齐 | 否 |
| 正文拉丁字母/数字 | Times New Roman | 小四 (12pt) | — | 否 |
| 摘要标题 | 黑体 | 四号 (14pt) | 左对齐 | 是 |
| 摘要正文 | 宋体 | 小四 (12pt) | 两端对齐 | 否 |
| 表格标题 | 黑体 | 五号 (10.5pt) | 居中 | 是 |
| 表格内容 | 宋体 | 五号 (10.5pt) | 居中 | 否 |
| 图片标题 | 黑体 | 五号 (10.5pt) | 居中 | 是 |

### 段落规范
- 行距：**1.5 倍行距**
- 首行缩进：2 字符（约 24pt for 小四号）
- 段前/段后：0（正文），适当间距（标题）

### 表格规范（三线表）
- 顶线：1.5pt 粗实线
- 表头下线：0.5pt 细实线
- 底线：1.5pt 粗实线
- 无竖线，数据行之间无横线

### 公式规范
- 使用 OMML (Word 原生 Math) 或 AxMath 渲染，不可用纯文本
- 公式居中显示，编号右对齐：(1), (2), ...
- 变量符号：Times New Roman 斜体；常量/单位：Times New Roman 正体

### 图片规范
- 分辨率 ≥ 300 DPI
- 图题：黑体 五号，居中，置于图下方
- 坐标轴标签清晰可读

### 参考文献格式
书籍：[编号] 作者, 书名, 出版地: 出版社, 出版年.
期刊：[编号] 作者, 论文名, 杂志名, 卷(期): 年, 起止页码.
网上：[编号] 作者, 资源标题, 网址, 访问时间（年月日）.
