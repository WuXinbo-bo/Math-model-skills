# S8: Paper Drafting -- M003 Experiment Review + Paper Writing

**Goal**: Comprehensively review experiment results and write the complete competition paper.
**Dependency**: S7.5 (unified_kernel.md)
**Execution Style**: Builder -> Proposer -> Critic -> Planner (M003 review), then Writer drafts.
**Meeting**: M003 (experiment review via sub-agents)
**Mode**: **Independent Sub-Agent Call (Platform-Agnostic)**

---

## Stage Principles (Not Negotiable)

1. **DO NOT skip any step.** All 7 steps must be executed.
2. **Figure D-A-C narrative is mandatory**: Generate D-A-C templates before writing.
3. **Every number must reference upstream artifacts**: No numbers or conclusions may be凭空 generated.
4. **Abstract must be written last**: Write all main sections first, then write the abstract.
5. **Anti-AI-trace awareness throughout**: Follow references/nature-writing-guide.md blacklist.
6. **Each number must have >= 3 dimensions of discussion**: sensitivity/physical meaning/baseline comparison/cross-question consistency/uncertainty.
