## Sync to Gate Workspace (MANDATORY)

Gate scripts use `resolve_workspace()` to locate artifacts. If `CUMCM_Workspace/` exists but S0 artifacts were created in the current directory, sync them:

```bash
python -c "
import shutil, os
ws = 'CUMCM_Workspace'
os.makedirs(ws, exist_ok=True)
synced = []
for d in ['inputs', 'state', 'outputs', 'data', 'methods', 'src']:
    src = d
    dst = os.path.join(ws, d)
    if os.path.isdir(src) and not os.path.isdir(dst):
        shutil.copytree(src, dst)
        synced.append(f'{src} -> {dst}')
    elif os.path.isdir(src) and os.path.isdir(dst):
        for root, dirs, files in os.walk(src):
            rel = os.path.relpath(root, src)
            dst_dir = os.path.join(dst, rel)
            os.makedirs(dst_dir, exist_ok=True)
            for f in files:
                sf = os.path.join(root, f)
                df = os.path.join(dst_dir, f)
                if not os.path.exists(df):
                    shutil.copy2(sf, df)
                    synced.append(f'{sf} -> {df}')
if synced:
    print(f'Synced {len(synced)} items to {ws}/')
    for s in synced:
        print(f'  {s}')
else:
    print('Already in sync or CUMCM_Workspace/ is the workspace root.')
"
```

**Sub-Agent Lifecycle** (important):
```
Create inspector sub-agent
  -> Send prompt + required files + write path
  -> Inspector directly writes to outputs/reviews/S0_review.md
  -> Immediately destroy inspector sub-agent
```

**Why destroy**:
- Avoid context contamination
- Ensure next inspector invocation is a fresh instance
- Conserve resources (some platforms limit sub-agent count)
